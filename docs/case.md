import uuid 是什麼？

一句話：

uuid 是 Python 內建模組，用來產生「全世界幾乎不會重複的唯一識別碼」。

為什麼系統要用它？

現在的 Case 是：

不是人建的

不是一開始就有申請號

可能來自 OCR、PDF、掃描檔

一開始資訊不完整

👉 那系統要怎麼「指著它說：就是這一案」？

答案：給它一個 系統內部 ID

這就是 uuid 在做的事。

為什麼不用申請號當 case_id？

這點超關鍵：

❌ 不能一開始用申請號

OCR 還沒讀到

文件可能是殘缺的

有些文件根本還沒 filing

❌ 申請號在現實中也會有問題

跨國（TW / US / EP）

同申請號在不同局會撞

格式不一致

👉 申請號是「法律識別」
👉 uuid 是「系統識別」

uuid 不是為了好看，
是為了讓系統在資料還不完整時，也能安全地指認一個物件。
